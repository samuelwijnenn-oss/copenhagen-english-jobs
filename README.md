## Quick Start

### Development

```bash
npm install
npm run dev
```

Visit `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Click "Deploy"

That's it! Your site will be live in ~2 minutes.

## Add Real Jobs

Currently using sample data. To add real jobs:

1. Replace the `sampleJobs` array in `src/App.jsx`
2. Or connect to a job API (Adzuna, The Muse, etc.)
3. Or set up a backend with Supabase
